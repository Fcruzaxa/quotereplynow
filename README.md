# QuoteReplyNow starter app

This is a single-file web app you can launch today.

## What it does
Generates:
- Customer quote message
- Follow-up text
- Review request

## How to launch free
1. Create a free GitHub account if you do not have one.
2. Create a new repository.
3. Upload index.html.
4. Turn on GitHub Pages or deploy the repo to Netlify/Cloudflare Pages/Vercel.
5. Create a Stripe Payment Link, Gumroad product, Payhip product, or Ko-fi button.
6. Replace `PASTE_YOUR_PAYMENT_LINK_HERE` in index.html with your checkout link.
7. Share the app in Facebook groups, Reddit communities, Craigslist service posts, local business groups, and direct messages to service providers.
